RF protocol

Select the RF protocol to use. Currently only the HobbyKing HKR3000 and the LANEBoysRC-4CH protocoles are supported.

Address

This 5-byte hexadecimal number defines the unique nRF24 address used by this model.
5 hex numbers separated by ':'

Hop channels

A sequence of 20 channels that define the frequency hopping algorythm. The transmitter switches to the next channel in the squence every 5 milliseconds.

Note: for HobbyKing receivers only use a linear sequence of hop channels in the range of 0..69!

20 numbers between 0 and 125

Hardware

Hardware inputs define what kind of input mechanism is connected to each of the individual I/O ports on the Headless transmitter circuit board.
Hardware inputs are usually only configured once when you build a new transmitter.

Logical inputs

Logical inputs are the input signals that are processed by the mixer to output servo information sent to the transmitter. A logical input utilizes one or more hardware inputs to generate a value that can be read by the mixer.
